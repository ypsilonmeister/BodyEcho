# **AR漢字アプリ：空間書字による認知特性支援コンセプト仕様書**

## **1\. 開発の背景と課題（認知特性）**

漢字の書字における認知特性と、従来の「紙と鉛筆による書写」が抱える構造的な課題を解決するための新しいアプローチとして定義します。

* **極端な凹凸（WISC検査）：**「言語理解123」と「処理速度81」のギャップ。頭の中の高度な思考（スーパーコンピューター）を、鉛筆という細い出力器官（ダイヤルアップ回線）で出力しようとすることで大渋滞（パニック）が発生している状態。  
* **ビジュアルシンカー：**漢字を記号の連続ではなく「一枚の絵」として捉える特性。  
* **視覚模倣と巧緻性の困難：**「目で見て感覚的に真似る」ことが苦手であり、一枚の複雑な絵を小さなマス目に順序立てて鉛筆で模写する作業は、ワーキングメモリを極度に圧迫する過酷なマルチタスクとなっている。

## **2\. コアとなる3つの革新的なメリット**

| メリット | 詳細と効果   |
| :---- | :---- |
| **①「微細運動」から「粗大運動」への変換** | 指先や手首の細かい制御（巧緻性）という負荷の高い作業から、体全体を使って空中に大きく字を書く「粗大運動」へ変換。脳の出力ルートが変わり、マス目等の制約（ブレーキ）が外れることで、思考スピードを落とさず出力可能になりパニックを回避する。 |
| **②「ボディイメージの弱さ」の視覚的補完** | 自分の体の位置や力加減が感覚的に掴みにくい課題に対し、腕の動きを「漢字の軌跡（光のエフェクトなど）」としてリアルタイムに視覚化。「こう動かせばこうなる」という身体感覚を外部から補完し、協調運動の自然なトレーニングとして機能させる。 |
| **③「苦痛なドリル」から「表現・アート」へ** | マス目に記号を押し込める「評価される苦行」から、何もない空間という大きなキャンバスに全身でダイナミックに描く「表現活動」「魔法」へと転換。モチベーションの源泉を「直さなきゃ」から「もっと描きたい」へと180度反転させる。 |

## **3\. 認知・神経科学的アプローチとしての意義**

* **運動記憶（マッスルメモリ）との統合：**空間に全身で大きく描くことで、漢字を「体の大きな動きの軌跡」という一つの塊（チャンク）として捉え直す。視覚情報だけでなく、筋肉や関節の動きという「運動記憶」と結びつくため、より強固に文字の形を記憶・定着させることができる。  
* **評価からの解放：**「できないことを突きつけられる」状態をテクノロジーでバイパスし、特性に合った「新しい出力の枠」を自作・提供する。

## **4\. 既存システム（BodyEcho）からの技術拡張イメージ**

すでに構築済みの BodyEcho（MediaPipe Poseベース）を拡張することで、シームレスな実装が可能です。

`【実装スペック・アイデア】`  
`- トラッキング対象の変更:`   
  `手首（WRIST）または人差し指（INDEX_FINGER）の座標の軌跡を保持・描画（残像効果の無期限延長）。`  
`- 直感的なジェスチャー操作（UI操作の排除）:`  
  `- 描画のON/OFF: 「手をグーに握っている間だけ線が引ける」または「人差し指を立てた時だけ引ける」`  
  `- キャンバスのクリア: 「両手をパンと合わせる（両手首の接近）」または「両手を大きくワイパーのように振る」`  
`- 描画エフェクト:`  
  `ただの黒い線ではなく、「光の軌跡」や「炎」など、アートとしての達成感を感じられるパーティクル表現を採用。`
